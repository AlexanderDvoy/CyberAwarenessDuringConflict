Cyber Awareness During Conflict Dashboard

An interactive cybersecurity awareness dashboard that visualizes common cyber risks affecting the public during periods of geopolitical tension or crisis.
This project demonstrates how cybersecurity risks such as phishing, fake alerts, misinformation, and weak cyber hygiene can be translated into clear visual insights using a data dashboard.
The goal is to show how cyber awareness and threat analysis concepts can be communicated through data visualization.
Project Demo
The dashboard provides a simple way to explore cyber risks and defensive actions through an interactive interface.
Features include:
Cyber risk overview dashboard
Interactive risk explorer
Risk likelihood vs impact visualization
Public cyber defense recommendations
Filters for category, audience, and risk level
Visual analytics using charts and heatmaps

Installation

Clone the repository:
git clone https://github.com/YOUR_USERNAME/appName.git
Navigate to the project folder:
cd cyber_awareness_dashboard
Install required libraries:
python -m pip install -r requirements.txt
Running the Dashboard
Start the Streamlit server:
python -m streamlit run app.py
Then open your browser:
http://localhost:8501
Key Cyber Risk Areas Visualized

The dashboard focuses on several public cyber risk categories:
- Phishing and impersonation attacks
- Fake emergency alerts
- Misinformation spreading through messaging platforms
- Password reuse and weak authentication
- Delayed device updates
- Defensive Recommendations
- The dashboard highlights simple but effective cyber hygiene actions:
- Verify emergency messages through official sources
- Enable multi-factor authentication
- Avoid urgent links in messages
- Keep devices and applications updated
- Be cautious when forwarding information
